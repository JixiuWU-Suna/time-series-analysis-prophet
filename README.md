# time-series-analysis-prophet
Time series forecasting using Prophet (CPI data)
This project applies time series analysis to forecast monthly Consumer Price Index of Transportation in U.S. City average for all urban consumers from January 1947 to February 2026 using Meta's Prophet model.

## Methods
- Data preprocessing
- Seasonal decomposition
- Prophet model fitting
- Forecasting and uncertainty intervals

## Tools Used
- R
- Prophet
- RMarkdown

## Key Results
- Identified upward trend in CPI
- Generated future forecasts with confidence intervals
